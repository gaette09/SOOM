# SOOM

SOOM은 수영, 사이클, 러닝 데이터를 기반으로 훈련 흐름을 분석하고 커뮤니티와 클럽 활동까지 연결하는 철인3종 피트니스 앱 프로토타입입니다.

## 현재 포함된 것

- 한국어 SwiftUI 앱
- 더미 하네스 기반 운동 데이터
- 홈, 분석, 기록, 피드, 클럽 탭
- 운동 상세 지도 레이어
- 심박, 페이스, 파워 그래프
- 스플릿, 심박 존, AI 해석 카드
- 피드 상세와 클럽 상세
- 디자인 가이드 문서

## 실행

Xcode에서 `SOOM.xcodeproj`를 열고 iPhone 시뮬레이터로 실행합니다.
