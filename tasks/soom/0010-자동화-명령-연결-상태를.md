# SOOM Codex Task

## Project
SOOM iOS App

## Goal
자동화 명령 연결 상태를 점검한다

## Background
이 작업은 SOOM 앱의 사용자 경험과 브랜드 일관성을 개선하기 위한 목적이다. 기존 기능을 유지하면서 최소 범위에서 화면 구조, 문구, spacing 또는 시각적 위계를 점검한다.

## Scope


## Requirements
1. 작업 목표와 직접 관련된 화면 또는 파일만 확인한다.
2. 기존 기능과 계산 로직은 변경하지 않는다.
3. SOOM의 차분한 브랜드 톤을 유지한다.
4. 과한 리팩토링이나 구조 변경은 하지 않는다.
5. 필요 시 최소 범위의 문구, spacing, 시각적 위계만 조정한다.
6. 변경 후 가능한 경우 xcodebuild test와 build를 실행한다.

## Constraints
- main 브랜치에 직접 커밋하지 않는다.
- 기존 기능을 삭제하지 않는다.
- 불필요한 리팩토링은 하지 않는다.
- 작업 범위를 벗어난 파일은 수정하지 않는다.

## Validation
- xcodebuild test 실행 가능하면 실행한다.
- xcodebuild build 실행 가능하면 실행한다.
- 실행하지 못한 경우 이유를 보고한다.
